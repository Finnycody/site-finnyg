Favicon Project

Favicon project

This icon is a 16 by 16 pixal rockstar games logo that I coded using Hexidecimal. Each pixle is coded in bits of 12, four bits for each color; Red, green, and blue, which means that the level of each color (RGB) would be coded using one hex character. One hex diget being coded by 4 binary digets.

Finn and Ian's Drawing app

Drawing app

This drawing app that I coded using snap, allows users to freely draw with the cursor as well as choose their own RGB settings using sliders. You are also able to draw polygons using the polygon button, which asks you how many sides you want the polygon to have and also where you want the polygon to be drawn. There will be more features added to the app in the future, however as of now, we are limited to free drawing and the polygon generator/button.
